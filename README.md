# Analyse de la régularité des TGV en France

## Contexte

Les retards ferroviaires ont un impact direct sur la satisfaction client et la performance opérationnelle.  
À partir des données ouvertes SNCF, j’ai analysé l’évolution mensuelle de la régularité des TGV afin d’identifier les tendances et les variations significatives.

---

## Objectif

L’objectif de ce projet est de :

- mesurer le taux de régularité mensuel
- observer les évolutions dans le temps
- comparer les volumes de circulation
- mettre en évidence les périodes plus sensibles

---

## Données

Source : données ouvertes SNCF  
Régularité mensuelle des TGV nationaux  

---

## Méthodologie

1. Import du fichier CSV
2. Nettoyage et formatage des dates
3. Calcul des indicateurs clés en SQL
4. Création d’un dashboard de suivi sur Tableau

---

## Indicateurs analysés

- Nombre total de trains
- Nombre de trains en retard
- Taux de régularité (%)
- Évolution mensuelle
